# 👘 ヒトノハゴロモ

アプリURL
[https://hitonohagoromo.onrender.com](https://hitonohagoromo.onrender.com)


**「服を記録することは、生活を大切にすること。」**

ヒトノハゴロモは、単なるファッション管理アプリではありません。
「いつ、誰と、どんな服で過ごしたか」を記録することで、「人間関係の悩み」と「日々の生活リズム（QOL）」を整える、生活密着型の服装管理アプリケーションです。

## 📖 サービス概要 

「あの人と前回会った時、何を着ていたっけ？」
「最近、一日中パジャマで過ごしてしまっているな...」

忙しい現代において、服装は単なる「おしゃれ」以上の意味を持ちます。
このアプリは、「時間帯ごとの着用記録」と「対人関係の記録」に特化することで、以下の2つの価値を提供します。

1.  **対人関係の円滑化**: 「会った人」と「服」を紐付けることで、大切な人との再会時に"二度目の装い被り"を回避します。
2.  **生活リズムの可視化**: 朝・昼・晩の着替えを記録することで、整容（身だしなみ）の習慣を整え、心の健康とQOL向上をサポートします。

## 💡 開発の背景・想い 

> **「身だしなみ（整容）は、その人の心身の健康に直結する」**

私は元医療職として多くの患者様と接する中で、この事実を痛感してきました。
服を着替えるという行為は、単なる外見の問題ではなく、自分自身の心を整え、社会や他者との関わり方を左右する大切なスイッチです。

しかし、忙しい日常の中では振り返る余裕がなく、「誰に何を着て行ったか」という記憶の曖昧さが小さなストレスになることもあります。

**「普通のファッションアプリではなく、QOLに寄り添うアプリを作りたい」**

そんな想いから、ヒトノハゴロモは生まれました。
服を通して日々の自分を見つめ直し、他者との関係をよりスムーズにする。誰もが心地よく過ごせる生活の一助となることを目指しています。

## 🌟 特徴と差別化 

既存のクローゼットアプリの多くが「コーディネート提案（今日何を着よう？）」や「タンスの肥やし管理」を主目的としているのに対し、本アプリは「人間関係」と「生活衛生」にフォーカスしています。

### 1. 人との繋がりを記録 
* **重複アラート機能**: 服と「会った人」をセットで記録。次に同じ人と会う予定を入れる際、過去に同じ服を着ていなかったかを自動でチェックし、気まずい思いを未然に防ぎます。
* **対人履歴**: 「〇〇さんと会った時の服装リスト」を瞬時に振り返ることができます。

### 2. 生活リズムの整容 
* **時間帯別記録**: 1日を「朝・昼・夕・夜」に区切り、細かく記録が可能。「朝、パジャマから着替えられたか？」といった、生活リズムやメンタルダウン兆候のセルフチェック（日記）としても活用できます。

### 3. 直感的なUI/UX 
* **世代を問わない「使いやすさ」**: 複雑なメニュー階層を排除し、大きなボタンと分かりやすいアイコン配置を徹底。ファッションアプリだからとオシャレな英語表記を避け、誰でもわかるボタン設定にしました。デジタル機器の操作に不安がある方でも、迷わず「見たまま」直感的に操作が可能です。
* **レスポンシブデザインの徹底**: 毎日の記録が負担にならないよう、タップ数を減らしたシンプルな画面設計を採用。スマホ・PCどちらでも快適に動作し、日々の生活リズムを無理なく記録し続けられます。

## 🚀 今後の展望 

生活の質（QOL）をさらに向上させるため、**衛生管理**の観点から以下の機能実装を予定しています。

* **お下着の管理機能**: 
    * 人に見せない部分こそ、衛生的なサイクル管理が必要です。
    * 購入日や着用回数からの「買い替え時期」通知機能などを検討中。
* **洗濯サイクル管理**:
    * 制服やスーツなど、毎日洗わない服の「洗濯までの着用回数」を可視化し、適切な衛生管理をサポートします。
